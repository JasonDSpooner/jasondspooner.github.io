1. AI & LLM tooling

  • Paperclip — AI-agent control plane that runs the sports-writing agents, fact-checker, and daily routines.
    • Our sports pipeline: jasondspooner.github.io/sports
  • OpenCode CLI — Free local text generation for the MLB/CFL/FIFA/Content Marketing writers.
  • Google Gemini CLI — Free local text generation; currently used by the fact-checker.
    • Live fact-check report: jasondspooner.github.io/sports/fact-check.html
  • Pollinations.ai — Featured-image generation for articles.
  • Hermes Agent (Nous Research) — Self-improving agent runtime used by TaskMan / Juan’s World.
  • Kimi / Moonshot AI — Conversational LLM fallback in Juan’s World app.
  • Ollama — Local LLaVA vision review for coloring-book image QC.
  • OpenAI — GPT/Codex models via OpenCode and Pollinations.
  • Anthropic Claude — Claude models via OpenCode / Pollinations.

2. Sports data & betting tools

  • MLB Stats API — Live Blue Jays schedule, scores, player stats, and home-run tracker data.
    • Our Blue Jays hub: jasondspooner.github.io/bluejays.html
  • WCLC Sport Select — Alberta sports-betting odds scraping.
  • Open-Meteo — Weather data for betting models and venue conditions.
  • ESPN — General sports data reference.
  • Action Network — Odds / betting market reference.
  • StatsBomb open-data — Foundation dataset for the FIFA 2026 Smart Bet system.
    • Our FIFA 2026 project: github.com/vote4arealclown/SBFGSfifa2026pre

3. Programming languages & runtimes

  • Python — Dominant language for analytics, scrapers, betting models, and AI pipelines.
  • Node.js — Website generators, adapter glue, and Puppeteer scrapers.
  • TypeScript — Next.js frontends and Paperclip UI.
  • GnuCOBOL — Retro terminal apps for baseball stats, betting, and expense tracking.
  • libpq (PostgreSQL C API) — Bridge COBOL to PostgreSQL.

4. Web frameworks & UI libraries

  • Next.js — React framework for SaskPoly, Juan’s World, and Jason’s Journal.
    • juansworld.xyz | saskpoly.xyz
  • React — UI library for most frontends.
  • FastAPI — Python API backend for SBFGS analytics.
  • Flask — Python web backend for TaskMan, Juan’s World, schedule app, and KDP tool.
  • Vite — Build tool for React SPAs.
  • Streamlit — Quick UI for the Amazon Nova copy/image panel.
  • Bootstrap 5 — Dashboard UI for SBFGS.
  • Tailwind CSS — Styling for Vercel sites.
  • Textual — Interactive terminal dashboards.
  • Rich — Terminal formatting and tables.

5. Databases & storage

  • PostgreSQL — Analytics database for Smart Bet systems.
  • SQLite — Local embedded database for apps.
  • Supabase — Auth + managed Postgres for Next.js apps.
  • Upstash Redis — Serverless Redis, rate limiting, and sessions.
  • Prisma — ORM for Supabase in SaskPoly.
  • Vercel KV — Serverless Redis for Jason’s Journal.
