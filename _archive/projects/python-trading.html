Python Trading Platform

Automated crypto futures trading on the Hyperliquid network using Python, technical analysis, and machine learning.

Back to Timeline

Overview

This project is a fully automated cryptocurrency trading platform built in Python, designed to trade futures on the Hyperliquid decentralized exchange. The system combines real-time market data, technical indicator analysis, and machine learning optimization to execute trades with minimal human intervention.

I developed this because I wanted to understand the intersection of algorithmic finance and decentralized infrastructure — and because manually watching charts 24/7 isn't sustainable.

What It Does

  • Connects to Hyperliquid's API for real-time order book and position data
  • Implements technical strategies using RSI, Bollinger Bands, and custom signals
  • Backtests strategies against historical data before going live
  • Uses machine learning to optimize strategy parameters over time
  • Manages risk through position sizing, stop-losses, and leverage controls

Current Configurations

  • Binance Top 50 Pairs, 20 Trade Max — Demo: Dry-run testing on Binance's top 50 pairs
  • Hyperliquid Top 50 Pairs, 20 Trade Max — Demo: Similar test on Hyperliquid for meme coins and stablecoins
  • Hyperliquid Top 7 Pairs, $100 Balance — Demo: Testing top 7 meme coin pairs with simulated $100 balance
  • Gold Trading via PAGX Token — Demo: Testing gold-tied token on Hyperliquid in dry-run mode
  • Hyperliquid Top 7 Pairs, $100 Balance — Live: Live test with real funds on Hyperliquid

Recent demo tests have shown a 95% win rate under controlled conditions.

Technology Stack

Python 3 Pandas / NumPy Hyperliquid API MetaMask / Web3 RSI & Bollinger Bands Scikit-learn Docker SQLite

Architecture

The bot runs as a containerized Python service. It fetches candlestick data, computes indicators, generates signals, and submits orders through Hyperliquid's L1 blockchain. All trades, balances, and signals are logged to a local database for post-trade analysis and strategy refinement.

Risk Warning

Crypto trading is inherently risky. Leverage amplifies both gains and losses. This platform is experimental and should not be considered financial advice. Only invest what you can afford to lose.

Results & Learnings

The most valuable output of this project hasn't been profit — it's been understanding. I've learned how decentralized order books work, how to manage API rate limits, how to avoid overfitting strategies to historical data, and how emotions (even automated ones) can be the biggest risk in trading.

The project is ongoing. I continue to iterate on strategy logic, add new pairs, and refine the ML optimization pipeline.
