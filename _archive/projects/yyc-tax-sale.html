2026 YYC Tax Sale Reporting

Real-time data platform tracking Calgary's 2026 property tax sale auction — over $31M in reserve value monitored daily.

Back to Timeline

The Problem

Calgary's annual tax sale auction involves hundreds of properties with reserve values, bids, and outcomes spread across fragmented PDFs, spreadsheets, and municipal websites. For investors, researchers, and real estate professionals, keeping track of what's available, what's sold, and at what price is a manual, error-prone process.

The Solution

I built an automated reporting platform that:

  • Scrapes and parses auction data from municipal sources in real time
  • Computes aggregate statistics — total reserve value, average lot size, sale rates
  • Generates daily HTML reports with sortable tables and summary metrics
  • Tracks properties across multiple auction days to show bidding trends
  • Alerts users to high-value or strategically interesting lots

Technology

Python Pandas BeautifulSoup HTML Generation Cron / Scheduling Data Visualization

The platform runs on a scheduled Python pipeline. Each morning, it fetches the latest auction data, cleans and normalizes it, computes statistics, and outputs a fresh HTML report. The entire process is hands-off after initial configuration.

What I Learned

This project reinforced the value of making public data actually usable. Municipal datasets are often technically "open" but practically inaccessible. By building a simple, automated pipeline, I turned a slog of PDF parsing into actionable intelligence for real estate investors.
